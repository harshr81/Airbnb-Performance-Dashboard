# Airbnb-Performance-Dashboard
📊 Global Airbnb Strategy: Performance & Insights Dashboard An interactive data visualization tool built to explore global rental trends—focusing on market dominance, service quality metrics, and host verification to drive business growth.


## Introduction

This project focuses on real analytical thinking to solve everyday business questions. I explored how to connect different pieces of information (Data Modeling) and used simple formulas (DAX) to uncover interesting stories. The dashboard looks at which cities lead the market, when people travel the most (seasonality), how prices compare to hotels, and what makes a guest feel safe (trust insights).

![Dashboard Page 1]()


## Skills Showcased

The dashboard was built using the following tools and technologies:

- **Power BI Desktop:** Main data visualization platform used for report creation.
- **⚙️ Power Query:** Data transformation and cleaning layer for reshaping and preparing the data.
- **📊 Core Charts:** Utilized **Column, Bar, Line,** and **Matrix Charts** to show the market share and monthly reviews by city.
- **🧮 DAX (Data Analysis Expressions):** Used for calculated measures and simple logic.
- **Buttons & Bookmarks:** Used **Buttons & Bookmarks** To create a seamless navigation experience between charts.
- **Data Modeling:** Relationships established among tables to enable smooth cross-filtering.
